== Description ==

The plugin helps you accept payments using Western Union Transfers.

== Installation ==

1. Upload 'woocommerce-westerm-union-gateway' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access 'admin.php?page=wc-settings&tab=checkout&section=wunion' in order to change the description and the details provided in the email
